#Tic Tac Toe Game

##Description of Tic Tac Toe
Tic Tac Toe is the game played with 2 player.
In this project , there are codes that edxplain how to play in the backend.

##Purpose
In this game, the  goal is for a single player to win by filling in the lanes horizontally diagonally and vertically.

##How It Works 
The code start with definition of game board, variable expression and players icons.
After that, Game board is printed to the console by print() method in order to play.
Then ,The player idenbtified "X" is provided to provide input for the game.
It is necessary to check horizontally,vertically and diagonally for deciding there's a winner or not.
If there is no winner and the game is still running the currentPlayer changes to "O".
